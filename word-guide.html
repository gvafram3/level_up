How to Use This Guide

Each section below gives you the exact steps to complete a Word task. Follow along with an actual document open in Word. Screenshots are described in detail so you know exactly what to look for.

Step 1

Using Styles for Headings

Why This Matters

Manually bolding text for headings breaks when you need a Table of Contents. Word Styles create consistent formatting and enable automatic TOCs.

Step 1.1 Open your document. Click anywhere in your main title text. Click in title
Step 1.2 Go to the Home tab (top ribbon). Look for the "Styles" section — you'll see boxes labeled "Normal", "Heading 1", etc. Home → Styles
Step 1.3 Click "Heading 1". Your title text will change to the Heading 1 format (larger, bold, different font). Click Heading 1
Step 1.4 For section headings (like "Introduction", "Methods"), click in that text and apply "Heading 2". Click Heading 2
Step 1.5 For sub-sections under those, use "Heading 3". Never use more than 3 heading levels in student work. Click Heading 3
✅

Test it worked: Your headings should look different from body text and have consistent formatting. If you click in a heading, the corresponding Heading style should be highlighted in the Styles section.

Step 2

Auto Table of Contents

Why This Matters

Manual TOCs break when you add content. Auto TOCs update with one click and show page numbers automatically.

Step 2.1 Place your cursor where you want the TOC (usually after title, before introduction). Click position
Step 2.2 Click the "References" tab in the top ribbon. Look for "Table of Contents" on the left side. References → TOC
Step 2.3 Click "Table of Contents" and choose "Automatic Table 1" or "Automatic Table 2" (both work fine). Automatic Table 1
Step 2.4 Word inserts a TOC showing all your Heading 1, 2, and 3 text with page numbers. TOC appears
Step 2.5 To update later: Right-click the TOC → "Update Field" → "Update entire table" → OK. Right-click → Update
⚠️

Common mistake: If your TOC is empty or missing headings, go back to Step 1 — your headings aren't using Word Styles, they're just manually formatted.

Step 3

Headers & Page Numbers

Step 3.1 Click the "Insert" tab. Look for "Header & Footer" section on the ribbon. Insert → Header
Step 3.2 Click "Header" → choose "Blank" (simplest option). A header area opens at the top of your page. Header → Blank
Step 3.3 Type your name, student ID, and course code in the header. Press Tab to move text to the right side. Type info
Step 3.4 For page numbers: Insert → Page Number → Bottom of Page → Plain Number 2 (center) or Plain Number 3 (right). Page Number → Bottom
Step 3.5 Click "Close Header and Footer" or double-click in the main document area to exit header/footer mode. Close Header
Step 4

Citations & References

Step 4.1 Click References tab → "Style" dropdown → choose "APA" (most common) or "Vancouver" (for medical courses). References → Style → APA
Step 4.2 To add a source: References → "Insert Citation" → "Add New Source". Insert Citation → Add New
Step 4.3 Choose source type (Book, Journal Article, Website). Fill in author, title, year, etc. Click OK. Fill source details
Step 4.4 Word inserts an in-text citation like (Smith, 2023). The source is saved for reuse. Citation inserted
Step 4.5 For bibliography: Place cursor at end → References → "Bibliography" → "References" (APA) or "Bibliography". Bibliography → References
Step 5

Proper Tables for Data

Step 5.1 Place cursor where you want the table. Insert → Table → drag to select size (e.g., 3 columns × 4 rows). Insert → Table → drag
Step 5.2 Click in the first row and type your column headers (e.g., "Course", "Credits", "Grade"). Type headers
Step 5.3 Select the header row → Table Design tab → "Header Row" checkbox (makes headers bold/different color). Table Design → Header Row
Step 5.4 Fill in your data in the remaining rows. Press Tab to move between cells. Fill data
Step 5.5 To add borders: Select entire table → Table Design → Borders dropdown → "All Borders". Borders → All Borders
Step 6

Track Changes for Group Work

Step 6.1 Before editing a shared document: Review tab → "Track Changes" button (it should highlight/turn on). Review → Track Changes
Step 6.2 Make your edits. Additions appear underlined, deletions appear crossed out, all in colored text. Edit normally
Step 6.3 To see who made changes: Review → "Show Markup" → ensure "Comments" and "Insertions and Deletions" are checked. Show Markup
Step 6.4 Before final submission: Review → "Accept" dropdown → "Accept All Changes" to make all edits permanent. Accept All Changes
Step 6.5 Turn off Track Changes before submitting: Review → "Track Changes" (button should no longer be highlighted). Turn off tracking
Step 7

Find & Replace

Step 7.1 Press Ctrl + H (Windows) or Cmd + H (Mac). The Find and Replace dialog opens. Ctrl + H
Step 7.2 In "Find what" box: type the text you want to replace (e.g., "colour"). Type find text
Step 7.3 In "Replace with" box: type the replacement text (e.g., "color"). Type replace text
Step 7.4 Click "Replace All" to change every instance, or "Find Next" then "Replace" to go one by one. Replace All
Step 7.5 Word shows "X replacements made". Click "Close" to finish. Close dialog
Step 8

Save as PDF

Step 8.1 File → "Save As" (or "Export" in some Word versions). File → Save As
Step 8.2 Choose your save location (Desktop, Documents, or your KNUST course folder). Choose location
Step 8.3 In "Save as type" dropdown: choose "PDF (*.pdf)" instead of "Word Document". Save as type → PDF
Step 8.4 Give it a proper name: CourseCode_Assignment_YourName_2024.pdf Name properly
Step 8.5 Click "Save". Word creates a PDF that looks identical on any device. Click Save
