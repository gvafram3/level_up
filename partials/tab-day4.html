MS Word Essentials

What Every Student Must Know

Styles & HeadingsUse Heading 1/2 styles instead of manually bolding text. Enables automatic Tables of Contents and consistent formatting across your entire document.Home → Styles
Auto Table of ContentsOnce headings are styled, insert a TOC that updates automatically when content changes. Required for all long reports.References → TOC
Headers & Page NumbersAdd your name and course code in the header. Page numbers in the footer. Standard academic document requirement.Insert → Header/Footer
Citations & ReferencesUse Word's reference manager to insert in-text citations and auto-generate a bibliography in APA or Vancouver format.References → Citations
Tables for DataUse proper Word tables with headers and borders for data. Never use manual spacing to fake a table.Insert → Table
Track ChangesUse for group assignment editing so everyone sees modifications. Accept/reject all changes before final submission.Review → Track Changes
Find & ReplaceReplace all instances of a word instantly. Fix consistent errors throughout a long document in seconds.Ctrl + H
Save as PDFAlways submit as PDF unless told otherwise. PDFs preserve formatting on all devices and operating systems.File → Save As → PDF
Digital Organisation

📁 Folder Structure Rule

Create: KNUST → [Year] → [Semester] → [Course Code]. Inside each course: Lectures / Assignments / Past Questions / Resources. Never leave files permanently in Downloads or Desktop.

📝 File Naming Convention

Always: [CourseCode]_Assignment1_YourName_2024.docx — not "final FINAL v3 copy.docx". Consistent naming saves hours of searching across a semester.

☁️ Cloud Backup (Non-Negotiable)

Sync your KNUST folder to Google Drive or OneDrive. Hard drives fail, phones get stolen, laptops vanish. Your work must be recoverable from any device, anywhere.

📋 Note-Taking Methods

Cornell Notes (notes / cues / summary), Mind Maps (visual, great for complex topics), Outline Method (hierarchical, great for lectures). Pick one per course and commit.
